# Python Interview Questions — 2-Day Plan

~38 questions across core concepts. ~19 per day.

---

## Day 1 — Fundamentals (~4–5 hours)

| Package | Questions | Focus |
|---------|-----------|-------|
| **decorator** | intro, measure_execution_time, log_function_args, memoization, repeat_n_times, args_kwargs_decorator | Measure time, log args, memoize, repeat N times, *args/**kwargs |
| **closure** | counter, multiply_fixed_multiplier | Counter, multiply with fixed factor |
| **generator** | fibonacci, read_file_line_by_line | Fibonacci, read file line by line |
| **iterator** | custom_iterator_class, iter_and_next | Custom iterator, `__iter__`/`__next__` |
| **magic_methods** | str_, repr_, len_, eq_, getitem | `__str__`, `__repr__`, `__len__`, `__eq__`, `__getitem__` |
| **class_oop** | singleton, immutable_object | Singleton, immutable object |
| **functional_programming** | map_manually, filter_manually, reduce_manually, loop_to_list_comprehension | map, filter, reduce, list comprehension |

---

## Day 2 — Applied & Interview (~4–5 hours)

| Package | Questions | Focus |
|---------|-----------|-------|
| **context_manager** | context_manager_class | Context manager with `__enter__`/`__exit__` |
| **advanced_internal** | lru_cache, custom_dictionary, custom_range | LRU cache, custom dict, custom range |
| **closure_scope** | fix_late_binding, nonlocal_modify_closure | Late binding, `nonlocal` |
| **real_interview_coding** | reverse_string, first_non_repeating_char, count_char_frequency, find_duplicates, flatten_nested_list, merge_two_dicts, check_anagrams, find_missing_number, stack_using_list, queue_using_list | Reverse string, first non-repeat, char freq, duplicates, flatten, merge dicts, anagrams, missing number, stack, queue |

---

## Suggested order

1. Start with `decorator/intro_to_decorators.py` and run it.
2. Do Day 1 packages in order (decorator → closure → generator → …).
3. Do Day 2 packages in order.
4. Use `real_interview_coding` last for practice.
